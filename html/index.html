Selamat Datang DI SPKMDA


SPK Penempatan Wali Kelas adalah sebuah inovasi sistem berbasis website yang dapat membantu User dalam pengambilan keputusan penempatan wali Kelas.
Pada sistem SPK ini diberikan beberapa fitur dengan fungsi tertentu.
1. Home, fitur ini digunakan untuk menampilkan tampilan awal dari sistem setelah User login ke dalam sistem
2. Kriteria, fitur ini berfungsi untuk mengelola data kriteria hingga mendapatkan bobot dari masing-masing kriteria dengan metode AHP. Lihat selengkapnya disini
3. Alternatif, fitur ini berfungsi untuk mengelola data Alternatif terkait dengan tambah, ubah dan hapus alternatif.
4. Proses, fitur ini berfungsi untuk olah data matriks dengan metode MOORA hingga mendapatkan hasil nilai perengkingan tiap alternatif. Lihat selengkapnya disini
5. Rekap, fitur ini berfungsi untuk mengelola data rekap hasil proses perhitungan dan rekap data pengguna.
